About This Project

The objective of this project was to create a blog for my little brother for him to showcase his poetry and short stories.

For this project, I wanted to find an easy to use Content Management System that can deliver content with Restful APIs and would allow me or my brother to control the content on the website dynamically. I came across an awesome service called Contentful. Contentful is a content management as a service (CaaS), it's very similar to WordPress, but without the PHP and MySQL. I also decided to build this project in React because it is efficient and makes writing JavaScript easier.

Technologies Used

  • HTML5
  • CSS3
  • JavaScript
  • NPM
  • Node.js
  • React.js
  • Contentful API
